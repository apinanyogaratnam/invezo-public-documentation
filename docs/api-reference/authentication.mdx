---

Getting your API Key To get started with the Invezo, you must first sign up for
an account [here](https://app.invezo.com/signup). After signing up and verifying
your email address, navigate to the
["API Key" section](https://app.invezo.com/settings/keys) of the settings page.

If you have problems with retrieving your API key, please email
[support@invezo.com](mailto:support@invezo.com) and we will promptly resolve
your issue.
