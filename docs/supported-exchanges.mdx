---
title: "Supported Exchanges"
---

The Invezo API supports the follow exchanges:

| Exchange   | Parameter   |
| ---------- | ----------- |
| Binance    | `binance`   |
| Binance US | `binanceus` |
| Coinbase   | `coinbase`  |
| FTX        | `ftx`       |
| Kraken     | `kraken`    |
| OKX        | `okx`       |
| Bitfinex   | `bitfinex`  |
| Bitstamp   | `bitstamp`  |
| GateIO     | `gateio`    |
